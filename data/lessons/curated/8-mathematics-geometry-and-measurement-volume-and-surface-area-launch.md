# Volume & Surface Area

## Learning Goal
- I can choose and use the correct volume formula for cylinders, cones, and spheres.
- I can compare the volumes of related solids and explain how the formulas are connected.
- I can distinguish volume formulas from surface-area reasoning so I do not apply the wrong measure.

## Introduction
By Grade 8, solid geometry includes figures that are more complex than rectangular prisms.

You may need to find the volume of:

- a cylinder
- a cone
- a sphere

These solids do not all use the same formula, but the formulas are related. A cone is connected to a cylinder, and a sphere has its own structure based on radius.

In this lesson, focus on three questions:

1. Which formula fits the solid?
2. What measurements does the formula require?
3. How can I check whether my answer makes geometric sense?

## Key Vocabulary
**Radius**: The distance from the center of a circle to its edge.

**Cylinder**: A solid with two parallel circular bases.

**Cone**: A solid with one circular base and one vertex.

**Sphere**: A perfectly round solid in which every point on the surface is the same distance from the center.

**Volume**: The amount of space inside a solid.

## Choosing the Correct Formula
Use these volume formulas:

- cylinder: `V = pi r^2 h`
- cone: `V = (1/3) pi r^2 h`
- sphere: `V = (4/3) pi r^3`

Notice the structure:

- cylinder and cone both use circular base area `pi r^2`
- the cone uses one-third of the matching cylinder volume
- the sphere depends only on radius

Before solving, identify:

1. the type of solid
2. the radius
3. the height, if the solid has one

## Worked Example 1: Volume of a Cylinder
A cylinder has radius `3` units and height `8` units.

Use:

`V = pi r^2 h`

Substitute:

`V = pi (3^2)(8)`

`V = pi (9)(8)`

`V = 72pi`

So the exact volume is:

`72pi cubic units`

As a decimal approximation:

`72pi` is about `226.2`

## Worked Example 2: Volume of a Cone
A cone has the same radius `3` units and height `8` units.

Use:

`V = (1/3) pi r^2 h`

Substitute:

`V = (1/3) pi (3^2)(8)`

`V = (1/3) pi (9)(8)`

`V = (1/3) 72pi`

`V = 24pi`

So the cone's volume is:

`24pi cubic units`

This confirms an important relationship:

The cone has one-third the volume of the cylinder with the same radius and height.

## Worked Example 3: Volume of a Sphere
A sphere has radius `4` units.

Use:

`V = (4/3) pi r^3`

Substitute:

`V = (4/3) pi (4^3)`

`V = (4/3) pi (64)`

`V = 256pi / 3`

So the exact volume is:

`256pi / 3 cubic units`

As a decimal approximation:

it is about `268.1`

The formula uses only radius because every dimension of the sphere depends on that one distance from the center.

## Worked Example 4: Comparing Related Solids
A cylinder and a cone each have:

- radius `5`
- height `12`

Find the cylinder volume.

`V = pi r^2 h`

`V = pi (25)(12) = 300pi`

Now find the cone volume.

`V = (1/3) pi r^2 h`

`V = (1/3)(300pi) = 100pi`

Interpretation:

The cone's volume is exactly one-third of the cylinder's volume when the radius and height match.

This is a useful reasonableness check in many problems.

## Surface Area Is Not the Same Measure
Even though the topic includes surface area, the formulas in this lesson are for volume.

That means the answer should be in:

- cubic units

not square units.

If a problem asks about:

- how much material covers the outside of the solid
- paint
- metal sheeting

then the problem is asking about surface area instead.

Always read the question before selecting the formula.

## Common Mistake: Forgetting to Square or Cube the Radius
Students often remember `r` but forget the exponent.

Example:

For a cylinder with radius `3` and height `8`, writing:

`V = pi (3)(8) = 24pi`

is incorrect because the radius should be squared.

Correct setup:

`V = pi (3^2)(8) = 72pi`

For a sphere, the radius must be cubed:

`r^3`, not `r^2`

These exponents are essential because the formulas model three-dimensional space.

## Summary
Grade 8 volume work uses formulas for cylinders, cones, and spheres.

To solve well:

1. identify the solid
2. choose the correct volume formula
3. substitute the radius and height carefully
4. keep track of exact values like `pi` and use cubic units

The best check is geometric sense: a cone should be smaller than the matching cylinder, and every volume answer should reflect the structure of the solid you started with.
