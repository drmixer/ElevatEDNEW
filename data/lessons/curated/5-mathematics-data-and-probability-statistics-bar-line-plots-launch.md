# Bar/Line Plots

## Learning Goal
- I can create a bar plot to show categorical data.
- I can create a line plot to show measurement data.
- I can compare data values and explain what a graph shows.

## Introduction
Graphs help readers organize information quickly.

A bar plot is useful when the data belongs to categories, such as favorite fruits or pet choices. A line plot is useful when the data shows repeated measurements, such as pencil lengths or jump distances.

Students need to recognize not only how to build each graph, but also why one graph type fits a data set better than another.

## Key Vocabulary
**Bar plot**: A graph that uses bars to show how many items belong in each category.

**Line plot**: A graph that uses X marks above a number line to show measurement data.

**Category**: A label used to group data, such as flavor or animal.

**Data**: Information collected from observations, measurements, or surveys.

## Worked Example 1: Reading a Bar Plot
Suppose a bar plot shows:

- apples: `6`
- bananas: `4`
- oranges: `2`

The tallest bar is apples, so apples are the most popular choice.

If we compare apples and bananas:

`6 - 4 = 2`

So `2` more students chose apples than bananas.

## Worked Example 2: Reading a Line Plot
Suppose pencil lengths are:

`5, 5, 6, 7`

On a line plot:

- two X marks go above `5`
- one X mark goes above `6`
- one X mark goes above `7`

That tells us `5` inches is the most common measurement in the set.

## Common Mistake
Students sometimes use a bar plot for measurement data that belongs on a line plot, or they forget that each X on a line plot stands for one value.

Ask:

- Is this a category or a measurement?
- Does each mark stand for one piece of data?

## Summary
Bar plots and line plots both organize data, but they serve different jobs.

1. Use bar plots for categories.
2. Use line plots for repeated measurements.
3. Read the graph carefully before making comparisons.
