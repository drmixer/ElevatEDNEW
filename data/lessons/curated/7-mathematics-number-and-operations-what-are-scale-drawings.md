# What are Scale Drawings?

## Learning Goal
- I can explain how a scale drawing represents a larger or smaller figure.
- I can use a scale factor to find missing side lengths.
- I can compare corresponding lengths to decide whether two drawings are scaled copies.

## Introduction
A scale drawing changes size but keeps shape.

That means every corresponding length changes by the same multiplier.

Scale drawings appear in many real situations:

- maps
- blueprints
- room plans
- model cars

In this lesson, focus on three questions:

1. What does the scale tell me?
2. How do I find an actual length from a drawing length?
3. How can I tell whether two figures are scaled copies?

## Key Vocabulary
**Scale drawing**: A drawing in which all lengths are multiplied by the same factor.

**Scale**: A ratio that compares drawing lengths to actual lengths.

**Scale factor**: The number used to enlarge or reduce every corresponding side.

**Corresponding sides**: Sides in the same relative position in two figures.

**Proportional**: Keeping the same multiplicative relationship.

## How Scale Drawings Work
If a drawing uses the scale `1 inch : 4 feet`, then every `1` inch in the drawing stands for `4` feet in real life.

The relationship must stay consistent for every side.

If one side uses a different multiplier, the figure is not a correct scale drawing.

## Worked Example 1: Find an Actual Length
On a blueprint, `1` centimeter represents `3` meters.

A hallway measures `6` centimeters on the drawing.

Multiply by the scale factor:

`6 x 3 = 18`

So the actual hallway length is `18` meters.

## Worked Example 2: Check for a Scaled Copy
Figure `A` has side lengths:

- `3`
- `5`

Figure `B` has side lengths:

- `6`
- `10`

Compare the ratios:

`6 / 3 = 2`

`10 / 5 = 2`

The same scale factor works for both sides, so Figure `B` is a scaled copy of Figure `A`.

## Common Mistake
Students sometimes add the same amount to every side instead of multiplying by the same factor.

Scale drawings use multiplication, not repeated addition.

For example:

- `3` to `6` multiplies by `2`
- `5` to `8` does not multiply by `2`

So those side lengths would not form a scaled copy.

## Summary
Scale drawings preserve shape by keeping all corresponding side lengths proportional.

To solve well:

1. identify the scale or scale factor
2. match corresponding sides
3. multiply or divide consistently
4. check that every ratio stays the same
