# Moves in the Plane

## Learning Goal
- I can describe translations, reflections, and rotations on the coordinate plane.
- I can predict how a point or figure moves under a rigid transformation.
- I can compare the original figure and the image to explain what stays the same.

## Introduction
Rigid transformations move a figure without changing its size or shape.

In Grade 8, the coordinate plane gives a precise way to describe those moves.

Instead of saying a figure "looks flipped" or "shifted over," you can name the exact transformation and track how each vertex changes.

In this lesson, focus on three questions:

1. How do translations, reflections, and rotations move points on the plane?
2. Which properties stay the same after a rigid transformation?
3. How can I use coordinates to check whether a move was described correctly?

## Key Vocabulary
**Rigid transformation**: A move that preserves side lengths and angle measures.

**Translation**: A slide in which every point moves the same distance and direction.

**Reflection**: A flip across a line.

**Rotation**: A turn around a fixed point.

**Image**: The new figure after a transformation.

## Why Coordinates Help
On a coordinate plane, each vertex has an exact address.

That makes it easier to:

- describe how far a figure moves
- compare original points with image points
- check whether lengths and angles stayed the same

Coordinates turn a visual move into a precise geometric argument.

## Worked Example 1: Translation
Point `A(2, 3)` is translated right `4` units and down `1` unit.

Add `4` to the `x`-coordinate:

`2 + 4 = 6`

Subtract `1` from the `y`-coordinate:

`3 - 1 = 2`

So the image is `A'(6, 2)`.

## Worked Example 2: Reflection Across the y-axis
Point `B(5, -2)` is reflected across the `y`-axis.

A reflection across the `y`-axis changes the sign of the `x`-coordinate and keeps the `y`-coordinate the same.

So:

`B(5, -2) -> B'(-5, -2)`

## Worked Example 3: Rotation
Point `C(3, 1)` is rotated `180` degrees about the origin.

For a `180` degree rotation about the origin:

`(x, y) -> (-x, -y)`

So:

`C(3, 1) -> C'(-3, -1)`

## Common Mistake
Students sometimes change only one coordinate when the move requires changing both.

Check the rule against the transformation:

- translation: add or subtract from coordinates
- reflection: usually one coordinate changes sign
- `180` degree rotation about the origin: both coordinates change sign

## Summary
Moves in the plane are easier to describe when you track each vertex with coordinates.

To solve well:

1. identify the transformation
2. apply the correct coordinate rule
3. compare the original figure and the image
4. check whether size and shape stayed the same
