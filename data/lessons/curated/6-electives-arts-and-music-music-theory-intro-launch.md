# Intro: Music Theory (intro)

## Learning Goal
- I can use the C major scale to build a simple triad.
- I can explain how tonic, subdominant, and dominant chords do different jobs in a progression.
- I can use chord evidence to decide which short progression sounds finished.

## Introduction
Imagine one short song ending stops on G, while another returns to C after the same opening chords. Even if you do not know the chord names yet, the ending that returns to C usually feels more complete. It sounds like the music arrived somewhere instead of stopping in the middle of a thought.

In this lesson, you will learn a few music-theory tools that explain why. You will use the C major scale to build simple chords, learn the different jobs those chords can do, and test which short progression sounds settled at the end.

## Key Vocabulary
**Scale**: A set of notes arranged in order that gives a song its note collection.

**Triad**: A three-note chord built by stacking every other note from a scale.

**Tonic**: The home chord or note that sounds settled in a key.

**Subdominant**: A chord that moves the music away from home without sounding as tense as the dominant.

**Dominant**: A chord that creates a strong pull back toward the tonic.

**Progression**: A sequence of chords played in order.

## A Key Starts With a Scale
In the key of C major, the scale is:

- C
- D
- E
- F
- G
- A
- B
- C

That scale gives us the note collection for the lesson. When musicians build chords in a key, they usually choose notes from that scale instead of random notes from outside it.

You can think of the scale as the set of building blocks available for the song. Once the note collection is clear, we can start combining notes into chords with a predictable pattern.

## Building a Triad
One useful pattern is "take every other note."

If you start on C and skip every other scale note, you get:

- C
- E
- G

That makes a C major triad.

If you start on F and use the same pattern, you get F-A-C. If you start on G, you get G-B-D. These are all triads because each one uses three notes stacked in the same every-other-note pattern.

Musicians also notice whether a triad sounds major or minor. A quick way to think about it is this: if the middle note is lowered by a half step, the chord changes from major to minor. In this lesson, we will stay with the major triads built on C, F, and G so the harmonic jobs stay easy to hear.

## Chord Jobs In A Progression
Chords do more than name note groups. They also do jobs inside a progression.

In C major, three especially useful chords are:

- **C major (I)**: tonic, or home
- **F major (IV)**: subdominant, which moves away from home
- **G major (V)**: dominant, which creates a strong pull back to home

That is why a progression can feel like it has motion. The tonic feels settled. The subdominant helps the music move. The dominant creates tension and expectation.

If a short song stops on the dominant chord, many listeners feel that something is missing. The dominant sounds like it wants one more move back to the tonic.

## Worked Example: Which Ending Sounds Finished?
Compare these two progressions:

- **Progression A**: C-F-G
- **Progression B**: C-F-G-C

A strong claim is that Progression B sounds more finished.

Here is the evidence and reasoning:

- **Evidence**: Progression B ends on C major, the tonic chord.
- **Reasoning**: The progression moves away from home through F major, creates tension with G major, and then returns to C major. That last return gives the ear a settled ending.

Progression A uses the same first three chords, but it stops on G major. Because G is the dominant, it sounds like the music is still leaning forward and waiting for the tonic to return.

## Guided Application: Build A Short Announcement Jingle
Suppose you want to create a four-chord background pattern for a short school announcement. The ending should sound calm and complete, not suspenseful.

A strong choice would be:

- C major
- F major
- G major
- C major

The Roman numeral pattern is:

- I
- IV
- V
- I

This works because the pattern begins at home, moves away, adds tension, and then returns home.

If you wanted the pattern to sound unfinished on purpose, you could stop on G major instead. That would leave the listener expecting one more chord.

When you analyze or build a progression, ask:

1. Which chord sounds like home?
2. Which chord creates the strongest pull?
3. Does the last chord sound settled or unfinished?

## Summary
Music theory helps explain why some note combinations and endings sound complete.

In this lesson, you learned that:

- the C major scale provides the note collection
- triads are built by taking every other note from the scale
- C major works like tonic, F major works like subdominant, and G major works like dominant
- a progression such as I-IV-V-I sounds finished because it returns to the tonic

The next time you hear a song ending, listen for the last chord and ask what job it is doing. If it sounds settled, it is probably acting like tonic. If it sounds like the music still wants to continue, it may be acting like dominant.
