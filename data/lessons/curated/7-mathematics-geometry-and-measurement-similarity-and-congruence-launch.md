# Similarity & Congruence

## Learning Goal
- I can tell whether two figures are congruent or similar.
- I can use corresponding side lengths to find a scale factor.
- I can explain how a scale drawing changes size while preserving shape.

## Introduction
In Grade 7 geometry, similarity and congruence connect directly to scale drawings.

If a drawing is an exact size match, the figures are congruent. If the drawing is larger or smaller by one constant multiplier, the figures are similar.

That distinction matters because scale drawings are designed to keep shape the same while changing size by a scale factor.

As you read, focus on two questions:

1. Are the matching side lengths equal or proportional?
2. If the figures are similar, what is the scale factor?

## Key Vocabulary
**Congruent**: Figures that have the same shape and the same size.

**Similar**: Figures that have the same shape, but not necessarily the same size.

**Corresponding sides**: Matching sides in the same relative position.

**Scale factor**: The constant ratio that enlarges or reduces a figure.

**Proportional**: Related by the same multiplicative ratio.

## Congruent Versus Similar
Use this check:

- if corresponding sides are equal and corresponding angles are equal, the figures are congruent
- if corresponding angles are equal and corresponding sides are proportional, the figures are similar

This matters in scale drawings because a scale drawing usually changes size but keeps the shape. That means it creates a similar figure, not a congruent one.

## Worked Example 1: Finding a Scale Factor
Triangle `A` has side lengths:

- `3`
- `5`
- `7`

Triangle `B` has side lengths:

- `6`
- `10`
- `14`

We want to decide whether the triangles are similar and, if so, find the scale factor from `A` to `B`.

Step 1: Match the corresponding sides.

- `3` with `6`
- `5` with `10`
- `7` with `14`

Step 2: Write the ratios in the same order.

- `6 / 3 = 2`
- `10 / 5 = 2`
- `14 / 7 = 2`

Step 3: Interpret the ratios.

All three ratios match, so the triangles are similar.

The scale factor from `A` to `B` is `2`.

## Worked Example 2: Scale Drawing Reasoning
A rectangular playground is shown on a scale drawing.

The drawing dimensions are:

- length `4` centimeters
- width `3` centimeters

The scale is:

`1 centimeter = 5 meters`

Step 1: Convert each length.

- actual length: `4 x 5 = 20` meters
- actual width: `3 x 5 = 15` meters

Step 2: Interpret what happened to the figure.

The actual playground has the same shape as the drawing, but each side is scaled by `5`.

So the drawing and the actual playground are similar, not congruent.

## Worked Example 3: Congruent Or Similar?
Polygon `P` has side lengths:

- `4`
- `6`
- `8`

Polygon `Q` has side lengths:

- `4`
- `6`
- `8`

Polygon `R` has side lengths:

- `6`
- `9`
- `12`

Compare `P` and `Q`.

All corresponding side lengths are equal, so they are congruent.

Compare `P` and `R`.

Check the ratios:

- `6 / 4 = 1.5`
- `9 / 6 = 1.5`
- `12 / 8 = 1.5`

The side lengths are proportional, so they are similar.

Because the lengths are not equal, they are not congruent.

## Why Corresponding Order Matters
The ratios only help if the matching sides are paired correctly.

Suppose a student compares:

- `6 / 4`
- `9 / 8`

These do not match because the wrong sides were paired.

When the order is inconsistent, the ratios can make similar figures look unrelated.

A strong solver always matches corresponding sides before dividing.

## Worked Example 4: Working Backward From a Scale Factor
Triangle `X` is similar to Triangle `Y`.

The scale factor from `X` to `Y` is `3`.

One side of Triangle `X` is `4`.

What is the corresponding side in Triangle `Y`?

Multiply by the scale factor:

`4 x 3 = 12`

So the corresponding side in Triangle `Y` is `12`.

If a side in Triangle `Y` were `15` instead, that side would not fit the same scale factor and the triangles would not be similar in the way described.

## Common Mistake: Calling Any Matching Shape Congruent
Students often see two figures with the same shape and call them congruent.

That misses the size question.

A scaled copy keeps shape but changes size.

Use this final check:

- exact same size -> congruent
- one constant scale change -> similar

## Summary
Grade 7 similarity and congruence depends on comparing corresponding parts carefully.

To solve well:

1. match the corresponding sides
2. check whether the side lengths are equal or proportional
3. find the scale factor when the figures are similar
4. explain whether the figures are congruent or similar and why

Scale drawings are one of the clearest examples of similarity because they preserve shape through one constant multiplier.
