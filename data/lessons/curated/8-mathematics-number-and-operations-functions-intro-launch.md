# Functions (intro)

## Learning Goal
- I can explain what makes a relation a function and why linear functions are useful.
- I can identify slope and initial value from a table, graph, equation, or context.
- I can compare proportional and non-proportional linear functions.

## Introduction
In Grade 8, introductory function work becomes more algebraic. You are not only checking whether a relation is a function. You are also analyzing how one quantity changes with another and how that change appears in equations, tables, graphs, and real situations.

A major focus is linear functions.

A linear function has:

- a constant rate of change
- a graph that is a straight line

Some linear functions are proportional and can be written as `y = kx`. Others are not proportional and include an initial value, such as `y = mx + b`.

In this lesson, focus on three questions:

1. How do I recognize a linear function?
2. What do slope and initial value mean in context?
3. How can I compare a proportional function with a non-proportional one?

## Key Vocabulary
**Function**: A relation in which each input has exactly one output.

**Linear function**: A function with a constant rate of change and a straight-line graph.

**Slope**: The rate of change in a linear relationship.

**Initial value**: The output when `x = 0`; in `y = mx + b`, this is `b`.

**Proportional relationship**: A linear relationship that can be written as `y = kx` and passes through the origin.

**Non-proportional relationship**: A linear relationship with a constant rate of change that does not pass through the origin.

## What Makes a Function Linear?
A relation is linear when equal changes in the input produce equal changes in the output.

Example table:

| x | y |
| --- | --- |
| 0 | 4 |
| 1 | 7 |
| 2 | 10 |
| 3 | 13 |

The output increases by `3` every time the input increases by `1`.

That constant change means the function is linear.

Its equation is:

`y = 3x + 4`

The slope is `3`, and the initial value is `4`.

## Worked Example 1: Finding Slope and Initial Value From a Table
Study the table:

| x | y |
| --- | --- |
| 0 | -2 |
| 1 | 1 |
| 2 | 4 |
| 3 | 7 |

Step 1: Find the change in the outputs.

The outputs increase by `3` each time.

So the slope is `3`.

Step 2: Find the output when `x = 0`.

That output is `-2`.

So the initial value is `-2`.

Step 3: Write the equation.

`y = 3x - 2`

Check with `x = 2`:

`y = 3(2) - 2 = 4`

The equation matches the table.

## Worked Example 2: Interpreting a Linear Function in Context
A streaming service charges a one-time sign-up fee of `$12` and `$5` each month.

Let `x` be the number of months and `y` be the total cost.

Write and interpret the function.

Equation:

`y = 5x + 12`

Interpretation:

- slope `5` means the cost increases by `$5` each month
- initial value `12` means the cost starts at `$12` before any monthly charges

Find the cost after `8` months:

`y = 5(8) + 12`

`y = 40 + 12`

`y = 52`

So the total cost is `$52`.

## Worked Example 3: Comparing Proportional and Non-Proportional Functions
Function A:

`y = 4x`

Function B:

`y = 4x + 6`

Compare them.

Step 1: Identify the slope.

Both functions have slope `4`.

So both increase at the same rate.

Step 2: Identify the initial value.

- Function A starts at `0`
- Function B starts at `6`

Step 3: Decide which is proportional.

Function A is proportional because it has the form `y = kx` and passes through `(0, 0)`.

Function B is not proportional because it has an added initial value.

This comparison matters because two functions can grow at the same rate but still represent different situations.

## Worked Example 4: Using a Graph to Identify a Function
A graph passes through the points:

- `(0, 3)`
- `(2, 7)`
- `(4, 11)`

Step 1: Check the changes.

From `(0, 3)` to `(2, 7)`:

- change in `x` is `2`
- change in `y` is `4`

So the slope is:

`4 / 2 = 2`

From `(2, 7)` to `(4, 11)`:

- change in `x` is `2`
- change in `y` is `4`

The rate stays constant, so the graph is linear.

Step 2: Identify the initial value.

When `x = 0`, `y = 3`.

So the equation is:

`y = 2x + 3`

## Function Language Across Representations
A strong Grade 8 solver can switch between forms:

- equation to table
- table to graph
- graph to verbal interpretation

For the equation `y = -1.5x + 9`:

- slope `-1.5` means the output decreases by `1.5` for each increase of `1` in the input
- initial value `9` means the graph starts at `(0, 9)`

That interpretation matters as much as the symbolic form.

## Common Mistake: Assuming Every Straight Pattern Is Proportional
Students sometimes see a straight-line graph or a linear equation and conclude that the relationship must be proportional.

That is not always true.

Example:

`y = 3x + 5`

This is linear, but it is not proportional because:

- it does not have the form `y = kx`
- its graph does not pass through the origin

To test proportionality, always check the initial value.

If the initial value is not `0`, the relationship is not proportional.

## Summary
Grade 8 functions focus on linear relationships, slope, and initial value.

To analyze a function well:

1. confirm that each input has one output
2. check whether the rate of change is constant
3. identify the slope and the initial value
4. decide whether the linear function is proportional or non-proportional

Functions become especially powerful when you can explain what the equation means in words, not just manipulate symbols.
