# Coordinate Geometry

## Learning Goal
- I can describe how translations, reflections, and rotations change coordinates.
- I can predict the image of a figure after a transformation on the coordinate plane.
- I can compare the original figure and the image to explain what stayed the same.

## Introduction
In Grade 8 coordinate geometry, the coordinate plane becomes a tool for studying transformations.

Instead of using coordinates only to locate points, you use them to describe how a figure moves:

- translation
- reflection
- rotation

The coordinate rules matter because they show exactly how each vertex changes and which properties stay unchanged.

In this lesson, focus on three questions:

1. How does each transformation change coordinates?
2. How can I find the image of a point or figure?
3. What properties stay the same after the transformation?

## Key Vocabulary
**Transformation**: A change in a figure's position or orientation on the plane.

**Translation**: A slide in which every point moves the same distance and direction.

**Reflection**: A flip across a line.

**Rotation**: A turn around a fixed point.

**Image**: The new figure after a transformation.

## What Transformations Do to Coordinates
Coordinate rules help describe transformations precisely.

Examples:

- translate right `3` and up `2`: add `3` to `x` and `2` to `y`
- reflect across the `x`-axis: keep `x`, change `y` to its opposite
- reflect across the `y`-axis: keep `y`, change `x` to its opposite
- rotate `180` degrees about the origin: change both coordinates to their opposites

These rules let you predict a full image without guessing from the picture.

## Worked Example 1: Translation
Point `A(2, 1)` is translated right `4` units and up `3` units.

Step 1: Update the `x`-coordinate.

`2 + 4 = 6`

Step 2: Update the `y`-coordinate.

`1 + 3 = 4`

So the image is:

`A'(6, 4)`

A translation keeps the figure's size and shape the same. It only changes location.

## Worked Example 2: Reflection Across the x-axis
Point `B(5, 3)` is reflected across the `x`-axis.

A reflection across the `x`-axis keeps the `x`-coordinate and changes the sign of the `y`-coordinate.

So:

`B(5, 3) -> B'(5, -3)`

Why?

The point moves the same distance below the `x`-axis that it was above the axis.

## Worked Example 3: Reflection Across the y-axis
Point `C(-4, 2)` is reflected across the `y`-axis.

A reflection across the `y`-axis changes the sign of the `x`-coordinate and keeps the `y`-coordinate the same.

So:

`C(-4, 2) -> C'(4, 2)`

This flips the point to the opposite side of the `y`-axis.

## Worked Example 4: Rotation 180 Degrees About the Origin
Triangle `PQR` has vertices:

- `P(1, 2)`
- `Q(4, 2)`
- `R(2, 5)`

Rotate the triangle `180` degrees about the origin.

For a `180` degree rotation about the origin:

`(x, y) -> (-x, -y)`

Apply the rule to each point:

- `P(1, 2) -> P'(-1, -2)`
- `Q(4, 2) -> Q'(-4, -2)`
- `R(2, 5) -> R'(-2, -5)`

The image has the same size and shape as the original triangle. Only its position and orientation change.

## What Stays the Same?
Translations, reflections, and rotations are rigid motions.

That means they preserve:

- side lengths
- angle measures
- shape

So if the original figure is a triangle, the image is still a congruent triangle.

Coordinates change, but the figure's size and shape do not.

This is one of the main ideas of Grade 8 transformation geometry.

## Common Mistake: Changing the Wrong Coordinate
Students often remember that "something changes sign" during a reflection but mix up which coordinate changes.

Example:

Reflect `(3, 5)` across the `x`-axis.

Wrong result:

`(-3, 5)`

Why is this wrong?

A reflection across the `x`-axis changes the vertical position, so the `y`-coordinate changes sign.

Correct result:

`(3, -5)`

To avoid mistakes, always connect the rule to the axis:

- `x`-axis affects up/down, so `y` changes
- `y`-axis affects left/right, so `x` changes

## Summary
Grade 8 coordinate geometry uses coordinates to describe transformations precisely.

To solve transformation problems well:

1. identify the type of transformation
2. apply the correct coordinate rule
3. find the image of each vertex
4. check what properties stayed the same

The power of coordinates is that they let you describe motion exactly, not just visually.
