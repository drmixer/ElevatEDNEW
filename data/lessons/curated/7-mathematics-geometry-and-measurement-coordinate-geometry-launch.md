# Coordinate Geometry

## Learning Goal
- I can use a coordinate grid to represent a scale drawing.
- I can apply a scale factor to compare a drawing with an actual figure.
- I can solve side-length problems by combining coordinate reasoning with scale reasoning.

## Introduction
In Grade 7, geometry often asks you to compare a drawing to a real object using scale.

A coordinate grid is useful here because it lets you place points exactly, measure horizontal and vertical distances clearly, and then apply a scale factor to move between the drawing and the actual object.

This means coordinate geometry is not just about plotting points. It is also about using coordinates to support proportional reasoning.

In this lesson, focus on three questions:

1. How do coordinates help describe a scale drawing?
2. How do I use a scale factor to compare drawing lengths and actual lengths?
3. How do I avoid confusing the drawing measurement with the real measurement?

## Key Vocabulary
**Scale drawing**: A drawing in which all lengths are proportional to the actual figure.

**Scale factor**: The number that tells how the drawing and actual figure are related.

**Coordinate plane**: A grid used to locate points with ordered pairs.

**Ordered pair**: Two numbers `(x, y)` naming a point's location.

**Proportional relationship**: A relationship in which two quantities are connected by the same multiplicative factor.

## Using Coordinates in a Scale Drawing
Suppose a garden is represented on a coordinate grid by the rectangle:

- `A(1, 1)`
- `B(7, 1)`
- `C(7, 4)`
- `D(1, 4)`

This is the drawing.

If the scale is:

`1 unit on the grid = 2 meters in real life`

then the coordinate lengths in the drawing can be converted into actual lengths.

The coordinate plane helps you find the drawing dimensions first. The scale factor helps you turn those dimensions into real ones.

## Worked Example 1: Finding Drawing Side Lengths
Use the rectangle above.

Find the drawing length of `AB`.

The segment is horizontal because both points have `y = 1`.

Subtract the `x`-coordinates:

`7 - 1 = 6`

So `AB` is `6` grid units long in the drawing.

Now find `BC`.

The segment is vertical because both points have `x = 7`.

Subtract the `y`-coordinates:

`4 - 1 = 3`

So `BC` is `3` grid units long in the drawing.

## Worked Example 2: Using the Scale Factor
The scale is:

`1 unit = 2 meters`

Convert the drawing dimensions:

- `AB = 6` units becomes `6 x 2 = 12` meters
- `BC = 3` units becomes `3 x 2 = 6` meters

So the actual garden dimensions are:

- length `12` meters
- width `6` meters

This is a proportional relationship because each drawing length is multiplied by the same factor of `2`.

## Worked Example 3: Working Backward from an Actual Length
A wall in a scale drawing is shown as a segment from `(2, 5)` to `(8, 5)`.

The scale is:

`1 unit = 4 feet`

What is the actual wall length?

Step 1: Find the drawing length.

The segment is horizontal, so subtract the `x`-coordinates:

`8 - 2 = 6`

Step 2: Apply the scale factor.

`6 x 4 = 24`

So the actual wall length is `24` feet.

## Worked Example 4: Finding a Missing Drawing Length
A rectangular room is `18` feet wide in real life.

The scale drawing uses:

`1 unit = 3 feet`

How wide should the room be on the grid?

Set up the relationship:

`drawing length x 3 = actual length`

`drawing length x 3 = 18`

`drawing length = 18 / 3 = 6`

So the room should be `6` grid units wide.

If one side is from `(2, 1)` to `(8, 1)`, that side would match the required drawing width because:

`8 - 2 = 6`

## Why Coordinates Help
Coordinates are useful in scale drawing because they make measurement exact.

Instead of estimating from a sketch, you can:

- read the endpoints
- subtract the matching coordinates
- then apply the scale factor

That keeps the reasoning organized:

1. find the drawing length
2. convert to the actual length

## Common Mistake: Adding Instead of Scaling
Students sometimes treat the scale like an amount to add.

Example:

If a segment is `5` units long and the scale is `1 unit = 3 feet`, a student may incorrectly write:

`5 + 3 = 8`

That is not how scale works.

Scale uses multiplication or division:

- drawing to actual: multiply
- actual to drawing: divide

Correct calculation:

`5 x 3 = 15` feet

## Summary
Grade 7 coordinate geometry can support scale-drawing problems by combining precise coordinates with proportional reasoning.

To solve well:

1. find the drawing length from the coordinates
2. identify the scale factor
3. multiply or divide depending on the direction of the problem
4. check whether the result makes sense in context

Coordinates tell you the drawing measurement. The scale factor tells you how that measurement connects to the real figure.
