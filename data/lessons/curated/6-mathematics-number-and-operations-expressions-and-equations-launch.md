# Expressions & Equations

## Learning Goal
- I can tell the difference between an expression, an equation, and an inequality.
- I can solve one-step equations using inverse operations.
- I can write and solve a one-step equation or inequality from a real-world situation.

## Introduction
Grade 6 algebra starts with a simple but important idea: a variable can stand for a number you do not know yet.

Sometimes you use a variable to describe a quantity. Sometimes you use it to solve for an unknown amount. The structure tells you which job the variable is doing.

In this lesson, pay attention to three questions:

1. Am I writing an expression, an equation, or an inequality?
2. Which operation is attached to the variable?
3. Which inverse operation will undo it?

## Key Vocabulary
**Expression**: A mathematical phrase with numbers, variables, and operations, but no equals sign.

**Equation**: A statement showing that two expressions are equal.

**Inequality**: A statement comparing two quantities with symbols such as `<`, `>`, `<=`, or `>=`.

**Variable**: A symbol that represents an unknown number.

**Inverse operations**: Operations that undo each other, such as addition and subtraction or multiplication and division.

**Solution**: A value that makes an equation or inequality true.

## Expressions Describe a Quantity
Suppose a museum charges `$8` for each student ticket and a one-time booking fee of `$3`.

If `s` is the number of student tickets, the expression

`8s + 3`

describes the total cost.

That expression does not ask a question yet. It only shows how the total depends on `s`.

Now suppose the class paid `$35` in all. Then the situation becomes the equation

`8s + 3 = 35`

If the class can spend at most `$35`, the situation becomes the inequality

`8s + 3 <= 35`

The same context can lead to three different forms depending on the question you are answering.

## Worked Example 1: Solving an Addition Equation
Solve:

`x + 7 = 19`

The variable has `7` added to it. To undo that, subtract `7` from both sides.

`x + 7 - 7 = 19 - 7`

`x = 12`

Check:

`12 + 7 = 19`

The solution is `12`.

## Worked Example 2: Solving a Multiplication Equation
Solve:

`5n = 40`

The variable is multiplied by `5`. To undo that, divide both sides by `5`.

`5n / 5 = 40 / 5`

`n = 8`

Check:

`5(8) = 40`

The solution is `8`.

This is the key pattern for one-step equations:

- identify the operation on the variable
- use the inverse operation
- keep the equation balanced

## Worked Example 3: Writing and Solving an Equation From Context
A club already raised `$14` from a bake sale. Each raffle ticket adds `$3`. The club has `$35` total. How many raffle tickets were sold?

Let `r` represent the number of raffle tickets.

Write the equation:

`14 + 3r = 35`

Subtract `14` from both sides:

`3r = 21`

Divide by `3`:

`r = 7`

Interpretation:

The club sold `7` raffle tickets.

Reasonableness check:

- `3 x 7 = 21`
- `21 + 14 = 35`

The answer fits the context.

## Worked Example 4: Solving an Inequality
A bus can hold at most `48` students. There are already `18` students on the bus. How many more students can get on?

Let `m` represent the number of additional students.

Write the inequality:

`18 + m <= 48`

Subtract `18` from both sides:

`m <= 30`

Interpretation:

At most `30` more students can board.

An inequality solution is different from an equation solution. It may describe many possible values instead of just one value.

## Common Mistake: Solving Without Naming the Operation
Students sometimes see

`5n = 40`

and subtract `5` because it is the number next to the variable.

That does not work because `5` is not being added. It is multiplying `n`.

A stronger method is:

1. identify the operation attached to the variable
2. choose the inverse of that operation
3. apply it to both sides
4. check the result in the original equation or inequality

Another common mistake is treating an inequality like it has only one answer. For `m <= 30`, values such as `12`, `25`, and `30` all work.

## Summary
Expressions, equations, and inequalities each have a different job.

- an expression describes a quantity
- an equation shows that two quantities are equal
- an inequality compares quantities when many values can work

To solve one-step equations and inequalities well:

1. identify the operation on the variable
2. use the inverse operation
3. keep both sides balanced
4. interpret the result in context

When you can write the correct form first, the solving step becomes much clearer.
