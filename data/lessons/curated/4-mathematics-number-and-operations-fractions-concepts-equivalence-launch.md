# Fractions (concepts, equivalence)

## Learning Goal
- I can use models to explain equivalent fractions.
- I can compare fractions with the same denominator.
- I can connect fraction bars to fraction notation.

## Introduction
Equivalent fractions look different, but they name the same amount.

Visual models make that idea easier to see.

## Key Vocabulary
**Equivalent fractions**: Fractions that name the same amount.

**Numerator**: The number of selected parts.

**Denominator**: The number of equal parts in the whole.

## Worked Example 1: Show Equivalence
Compare `1/2` and `2/4`.

If one bar is split into `2` equal parts and another is split into `4` equal parts, shading half the bar gives the same total amount.

So:

`1/2 = 2/4`

## Worked Example 2: Compare Same Denominator
Which is greater: `3/8` or `5/8`?

The denominators are the same, so compare the numerators.

Because `5` is greater than `3`, `5/8` is greater.

## Common Mistake
Students sometimes think a larger denominator means a larger fraction.

But when one whole is cut into more equal parts, each part is actually smaller.

## Summary
Equivalent fractions are easier to understand with visuals.

To reason well:

1. compare equal wholes
2. check the shaded amount
3. use models to justify equivalence
