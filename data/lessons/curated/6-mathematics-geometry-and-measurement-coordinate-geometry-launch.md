# Coordinate Geometry

## Learning Goal
- I can plot ordered pairs in the first quadrant of the coordinate plane.
- I can use coordinates to draw polygons and identify their vertices.
- I can find horizontal and vertical side lengths by comparing coordinates.

## Introduction
Coordinate geometry gives every point on a grid an exact location.

In Grade 6, the coordinate plane helps you do more than place points. It helps you build shapes and measure parts of those shapes using the coordinates of their vertices.

This is useful because a diagram can show what a shape looks like, but coordinates tell you exactly where the shape is and how far key points are apart.

In this lesson, focus on three questions:

1. How do I plot an ordered pair correctly?
2. How do coordinates describe a polygon?
3. How can I find horizontal and vertical lengths from the coordinates?

## Key Vocabulary
**Coordinate plane**: A grid formed by a horizontal axis and a vertical axis.

**Ordered pair**: Two numbers written as `(x, y)` that name a point's location.

**x-coordinate**: The first number in an ordered pair; it tells horizontal position.

**y-coordinate**: The second number in an ordered pair; it tells vertical position.

**Vertex**: A corner point of a polygon.

## Reading and Plotting Ordered Pairs
An ordered pair such as `(4, 3)` means:

- move `4` units to the right on the `x`-axis
- then move `3` units up on the `y`-axis

The order matters.

`(4, 3)` and `(3, 4)` are different points because they switch the horizontal and vertical positions.

In Grade 6 coordinate work, points are usually plotted in the first quadrant, where both coordinates are positive.

## Worked Example 1: Plotting a Point
Plot the point `(5, 2)`.

Step 1: Read the `x`-coordinate.

The first number is `5`, so move `5` units to the right.

Step 2: Read the `y`-coordinate.

The second number is `2`, so move `2` units up.

Step 3: Mark the point.

The point is located at `(5, 2)`.

Check:

- horizontal position is `5`
- vertical position is `2`

## Worked Example 2: Drawing a Polygon
Plot the points:

- `A(1, 1)`
- `B(5, 1)`
- `C(5, 4)`
- `D(1, 4)`

Then connect them in order.

What shape appears?

The figure is a rectangle.

Why?

- `AB` and `CD` are horizontal
- `BC` and `DA` are vertical
- opposite sides have matching lengths

The coordinates describe the shape even before you see the drawing.

## Worked Example 3: Finding Side Lengths
Use the rectangle above.

Find the length of `AB`.

The coordinates are:

- `A(1, 1)`
- `B(5, 1)`

The `y`-coordinates are the same, so the segment is horizontal.

Subtract the `x`-coordinates:

`5 - 1 = 4`

So the length of `AB` is `4` units.

Now find the length of `BC`.

- `B(5, 1)`
- `C(5, 4)`

The `x`-coordinates are the same, so the segment is vertical.

Subtract the `y`-coordinates:

`4 - 1 = 3`

So the length of `BC` is `3` units.

## Using Coordinates to Describe Shapes
Coordinates can tell you several things quickly:

- where a shape is located
- whether sides are horizontal or vertical
- how long horizontal and vertical sides are

For example:

If two points have the same `y`-coordinate, the segment between them is horizontal.

If two points have the same `x`-coordinate, the segment between them is vertical.

This makes side-length questions much easier.

## Worked Example 4: Missing Vertex Reasoning
Three vertices of a rectangle are:

- `P(2, 2)`
- `Q(7, 2)`
- `R(7, 6)`

What is the fourth vertex?

Look at the pattern:

- `P` and `Q` share `y = 2`
- `Q` and `R` share `x = 7`

To complete the rectangle, the missing point must:

- line up vertically with `P`, so `x = 2`
- line up horizontally with `R`, so `y = 6`

So the missing vertex is:

`(2, 6)`

## Common Mistake: Reversing the Coordinates
Students often confuse the order of the pair.

Example:

A student plots `(2, 5)` by moving up `2` and right `5`.

That produces the point `(5, 2)` instead.

To avoid this:

1. read the `x`-coordinate first
2. move horizontally first
3. read the `y`-coordinate second
4. move vertically second

The order in "ordered pair" is not optional.

## Summary
Coordinate geometry in Grade 6 helps you locate points and describe shapes precisely.

To solve coordinate problems well:

1. plot ordered pairs in the correct order
2. connect vertices carefully to form the polygon
3. compare `x`-coordinates for horizontal lengths
4. compare `y`-coordinates for vertical lengths

Coordinates turn pictures into exact mathematical information, which makes shapes easier to analyze and explain.
