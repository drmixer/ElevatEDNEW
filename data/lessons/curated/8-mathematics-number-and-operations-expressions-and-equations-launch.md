# Systems of Linear Equations

## Learning Goal
- I can explain what it means for an ordered pair to solve a system of linear equations.
- I can solve a system using graphing, substitution, or elimination.
- I can interpret whether a system has one solution, no solution, or infinitely many solutions.

## Introduction
One linear equation describes one relationship. A system of linear equations compares two relationships at the same time.

That matters in real situations because many problems involve two conditions that must both be true. A point is not a solution just because it works in one equation. It must satisfy both equations in the system.

When you solve a system, you are looking for the value pair where the two relationships agree.

In this lesson, focus on three ideas:

1. What does the solution mean on a graph and in an equation?
2. Which solving method fits the way the system is written?
3. How can I tell whether the system has one solution, no solution, or infinitely many solutions?

## Key Vocabulary
**System of linear equations**: Two or more linear equations considered together.

**Solution to a system**: An ordered pair that makes every equation in the system true.

**Intersection**: The point where two graphs cross.

**Substitution**: A method in which one variable expression is replaced with an equivalent expression from another equation.

**Elimination**: A method in which equations are added or subtracted to remove one variable.

**No solution**: A system in which the lines are parallel and never intersect.

## What a System Represents
Consider the system:

`y = x + 1`

`y = -x + 5`

Each equation describes a line.

The solution to the system is the point that lies on both lines. On a graph, that is the intersection. In algebra, it is the ordered pair that makes both equations true.

This is the main idea to keep in view:

- one equation alone gives many possible points
- a system narrows those choices
- the solution must satisfy both conditions at once

## Worked Example 1: Solving by Graphing
Solve the system:

`y = x + 1`

`y = -x + 5`

Because both equations are already in slope-intercept form, graphing is a natural method.

Use a few easy points.

For `y = x + 1`:

- `(0, 1)`
- `(2, 3)`

For `y = -x + 5`:

- `(0, 5)`
- `(2, 3)`

The lines intersect at `(2, 3)`.

Check the point in both equations:

First equation:

`3 = 2 + 1`

Second equation:

`3 = -2 + 5`

Both statements are true, so the solution is `(2, 3)`.

Graphing helps when the intersection lands on a clear point, but if the intersection is not easy to read exactly, substitution or elimination is usually stronger.

## Worked Example 2: Solving by Substitution
Solve the system:

`y = 2x - 1`

`x + y = 8`

The first equation already isolates `y`, so substitute `2x - 1` for `y` in the second equation.

`x + (2x - 1) = 8`

Combine like terms:

`3x - 1 = 8`

Add `1` to both sides:

`3x = 9`

Divide by `3`:

`x = 3`

Now substitute back into `y = 2x - 1`:

`y = 2(3) - 1 = 5`

So the solution is `(3, 5)`.

Check in the second equation:

`3 + 5 = 8`

The point works in both equations.

Substitution is efficient when one equation already gives a variable by itself or can be rearranged easily.

## Worked Example 3: Solving by Elimination
Solve the system:

`2x + y = 11`

`3x - y = 9`

The coefficients of `y` are opposites, so elimination is efficient.

Add the equations:

`(2x + y) + (3x - y) = 11 + 9`

`5x = 20`

`x = 4`

Substitute `x = 4` into `2x + y = 11`:

`2(4) + y = 11`

`8 + y = 11`

`y = 3`

So the solution is `(4, 3)`.

Interpretation:

The pair `(4, 3)` is the only ordered pair that makes both linear relationships true at the same time.

## Special Cases: No Solution and Infinitely Many Solutions
Not every system has exactly one solution.

Example with no solution:

`y = 2x + 1`

`y = 2x - 3`

These lines have the same slope but different intercepts. They are parallel, so they never intersect. The system has no solution.

Example with infinitely many solutions:

`2x + y = 8`

`4x + 2y = 16`

The second equation is just double the first one. They represent the same line, so every point on that line solves both equations.

A strong solver does not stop after manipulating symbols. They also ask what the equations mean geometrically.

## Common Mistake: Checking a Point in Only One Equation
Students sometimes solve for one variable, find an ordered pair, and stop without checking both equations.

That is not enough.

A system solution must satisfy every equation in the system.

Another common mistake is eliminating incorrectly by combining terms with the same sign when subtraction was needed. Before adding or subtracting equations, check which variable you want to remove and whether the coefficients are opposites or equal.

Use this checklist:

1. choose a method that fits the system's form
2. solve carefully for one variable or eliminate one variable
3. substitute back to find the second value
4. test the ordered pair in both original equations

## Summary
Systems of linear equations compare two linear relationships at once. Their solution is the ordered pair where the relationships agree.

To solve well:

1. use graphing for clear visual intersections
2. use substitution when a variable is already isolated
3. use elimination when coefficients can cancel efficiently
4. interpret the result as one solution, no solution, or infinitely many solutions

The key idea is not just solving for `x` and `y`. It is understanding why one ordered pair can satisfy two equations at the same time.
