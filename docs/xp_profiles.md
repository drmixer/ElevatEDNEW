# XP Profiles for ElevatED

These presets keep XP tuning simple without code changes. Each profile sets the four XP-related `platform_config` keys:

- `xp.multiplier`
- `xp.difficulty_bonus_multiplier`
- `xp.accuracy_bonus_multiplier`
- `xp.streak_bonus_multiplier`

## Preset values
- **Conservative:** multiplier 0.9, difficulty bonus 0.9, accuracy bonus 0.9, streak bonus 0.85.
- **Standard:** multiplier 1.0, difficulty bonus 1.0, accuracy bonus 1.0, streak bonus 1.0.
- **Boosted:** multiplier 1.2, difficulty bonus 1.1, accuracy bonus 1.1, streak bonus 1.15.

## How to apply
- From the Admin dashboard, use the new **XP profiles** buttons in Config Toggles to apply a preset in one click.
- To edit manually, use the Config key/value form with the keys above; changes take effect on the next config fetch.
